登录规则

登录规则表示用户登录边缘管理系统需要满足的约束条件，包括登录时间段、登录IP网段、登录MAC地址等。单个规则内部的约束条件必须同时满足，多个规则中有一个规则满足约束条件即可。

新增登录规则

  1. 在主菜单中选择“设置 > 安全策略 > 登录规则”。

    图1 登录规则

  2. 单击“新增规则”。

    图2 新增登录规则

  3. 根据表1提供的参数信息，设置系统登录规则。

    某条登录规则参数为空，规则状态为“启用”并保存时，表示此项登录无限制。

  4. 单击“确认”。

    界面上方弹出“保存成功”提示框，表示新增登录规则成功。

修改登录规则

  1. 在主菜单中选择“设置 > 安全策略 > 登录规则”。

    图3 登录规则

  2. 单击待修改规则右侧的“修改”。

    图4 修改登录规则

  3. 根据表1提供的参数信息，修改系统登录规则。
  4. 单击“确认”。

    界面上方弹出“保存成功”提示框，表示修改登录规则成功。

删除登录规则

  1. 在主菜单中选择“设置 > 安全策略 > 登录规则”。

    图5 登录规则

  2. 单击待删除规则右侧的“删除”。

    图6 删除登录规则

  3. 在“当前用户密码”文本框中输入系统当前的用户密码。
  4. 单击“确认”。

    界面上方弹出“删除成功”提示框，表示删除登录规则成功。

修改登录规则状态

  1. 在主菜单中选择“设置 > 安全策略 > 登录规则”。

    图7 登录规则

  2. 单击待启用规则右侧的“修改”，修改登录规则状态。

    • 单击使其变为，表示停用该规则。
    • 单击使其变为，表示启用该规则。
    图8 修改登录规则

    某条登录规则参数为空，规则状态为“启用”并保存时，表示此项登录无限制。

  3. 在“当前用户密码”文本框中输入系统当前的用户密码。
  4. 单击“确认”。

    界面右上角弹出“登录规则状态切换成功”提示框，表示修改登录规则成功。

导出登录规则

  1. 在主菜单中选择“设置 > 安全策略 > 登录规则”。

    图9 登录规则

  2. 单击“导出”，将当前系统的登录规则保存至本地。

    图10 导出规则

    不允许用户修改导出的文件，否则造成该文件不可用。

导入登录规则

登录规则不支持自行构造，只支持Web在线导出登录规则，迁移导入其他设备使用。

  1. 在主菜单中选择“设置 > 安全策略 > 登录规则”。

    图11 登录规则

  2. 单击“导入”，弹出“导入登录规则”对话框。

    图12 导入登录规则

  3. 在规则文件后单击，选择文件。上传10K以内的.ini文件。

    • 导入文件后会覆盖系统原有的登录规则。
    • 导入的文件名长度为1~255，可由大小写字母（a~z、A~Z）、数字（0~9）和其他字符（_.-）组成，且不能包含“..”且后缀为“ini”。
    • 导入的文件中最多可允许配置30条登录规则。

  4. 在“当前用户密码”文本框中输入系统当前的用户密码。
  5. 单击“确认”。

    界面上方弹出“登录规则导入成功！”提示框，表示导入登录规则成功。

参数说明

表1 登录规则

参数名称

参数说明

登录时间段

允许用户登录服务器的时间段。支持如下格式：

xx:xx表示允许用户每日登录的时间段，如起始时间为06:00，结束时间为12:00。

说明：
  • 同一条规则的起始时间和结束时间的格式必须保持一致。
  • 同一条规则的起始时间和结束时间必须同时配置，不可只配起始时间或结束时间。

登录IP网段
允许用户登录的具体IP地址或IP网段。支持IPV4地址，支持如下两种格式：
  • xxx.xxx.xxx.xxx：允许登录系统的单个用户的IP地址。
  • xxx.xxx.xxx.xxx/mask：允许登录系统的用户IP网段，其中“mask”为子网掩码长度，取值范围为1～32。

登录MAC地址
允许用户登录的具体MAC地址或MAC地址头。支持如下两种格式：
  • xx:xx:xx:xx:xx:xx：允许登录系统的单个用户的MAC地址。
  • xx:xx:xx：允许登录系统的用户MAC地址头。

启用状态
是否启用登录规则。可单击“启用状态”后的，筛选登录规则的状态。可选的状态参数如下：
  • 启用
  • 停止
说明：

筛选成功后，可通过“重置”按键，清空筛选内容，恢复到筛选前。

若登录规则启用状态为“打开”，且规则参数为空，表示此项登录无限制。

当前用户密码

输入当前用户密码。
