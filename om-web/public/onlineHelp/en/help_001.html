Welcome to Edge Management System Online Help

Edge Management System is a piece of system monitoring and management software for edge devices.

The edge management system has the following features:

  • Optimal user experience

    The software provides a visualized and easy-to-use graphical user interface (GUI) for you to perform interactive operations on edge devices.

  • Efficient management and maintenance

    The software provides remote management and hardware monitoring functions, which enable you to remotely access edge devices and monitor and manage device running status.

  • Secure access

    The software provides various management interfaces, which use encryption algorithms to ensure secure system access.

This document covers device alarm monitoring, fault locating, system management, data configuration, and parameter description.

You can log in to the WebUI or CLI of the edge management system. The figures provided may be different from the actual figures, but user operations are not affected. The actual information displayed on the WebUI of the edge management system may vary.
