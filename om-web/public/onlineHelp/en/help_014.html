Storage Configuration

This section describes how to manage the internal and external storage of an edge device.

The WebUI or FusionDirector provides a storage management page for users to perform the following operations:
  • Query information, including the device model, capacity, partition status, and file system format, about all storage devices, such as built-in eMMC flash modules, SD cards, M.2 SSDs, HDDs, and USB flash drives.
  • Create drive partitions and perform corresponding operations on drive partitions.

  • NFS Management
