Changing User Password

Procedure

  1. Move the mouse pointer to on the upper right corner of the page.
  2. Click Modify Password and set a new password.

    Figure 1 Changing user information
    Figure 2 Changing user password
    A password must meet the following requirements:

    • Contains 8 to 20 characters.

    • Contains at least three types of the following characters:

      • Lowercase letters: a to z

      • Uppercase letters: A to Z

      • Digits 0 to 9

      • Special characters: `~!@#$%^&*()-_=+\|[{}];:'",<.>/? and space
    • Cannot be the same as the user name or user name in reverse order.
    • Must be different from the last five passwords and cannot be the default password.

  3. Click Confirm. The login page is displayed.

    A dialog box is displayed, indicating that the modification is successful.
