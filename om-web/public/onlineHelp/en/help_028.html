Security Policy

Choose Settings > Security Policy. Then, you can:

  • View and set the timeout interval of the edge management system.

  • View and set the password validity period for the edge management system user.

  • View and manage user login security rules of the edge management system.

  • Import or export the weak password dictionary of the edge management system.

  • Import, replace, and query the SSL server certificate.

  • View and set the number of days to generate an alarm in advance before the edge management system certificate expires.

For security purposes, you are advised to change the IP address listened by Nginx to a fixed IP address and disable Nginx from listening to IPv6 addresses after Nginx is successfully deployed.
