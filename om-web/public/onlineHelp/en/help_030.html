Password Validity

Setting the Password Validity Period

  1. Choose Settings > Security Policy > Password Validity.

    Figure 1 Password validity

  2. Set the number of days in the validity period.

    You can click the list box to manually set the validity period.

    Specifically, you can click to set the validity period. One day is added each time you click , and one day is reduced each time you click .

    Figure 2 Setting the password validity

    The value ranges from 0 to 365, in days. The default value is 90 days. If the value is 0, the password is valid permanently. Using the same password for a long time increases the risk of brute force cracking and malicious guessing.

    For security purposes, set a proper validity period and change the password periodically.

  3. In the Current User Password text box, enter the password of the current user.
  4. Click Save.

    The message "Save successfully" is displayed, indicating that the password validity is set successfully.
