Home Page

The edge management system home page displays the system running status, basic information, temperature, power consumption, alarms, and CPU utilization, memory utilization, and NPU utilization of an edge device.

Figure 1 Home page
Table 1 Home page

Name

Description

Menu

Serves as the edge management system main menu and provides entries for management and settings.

User name

Provides entries for changing user passwords and logging out.

System Information

Displays the hostname, uptime, OS, firmware version, product asset label, serial number, and NPU version.

Alarms

Displays the running status (critical, major, or minor) of an edge device.

Temperature

Displays the temperature of the AI accelerator module.

Edge Station Information

Displays the CPU utilization, memory utilization, and NPU utilization.
