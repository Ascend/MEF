System Information

Scenarios

View the basic information and peripherals of the system.

Procedure

  1. Choose Management > System Information.

    Figure 1 System Information

  2. Check system information.

    • View the basic information. Table 1 describes the parameters.
      Table 1 Basic information parameters

      Parameter

      Description

      Hostname

      Indicates the hostname of the management system. You can click Edit to modify it.

      The value is a string of 1 to 64 characters.

      Letters, digits, and hyphens (-) are allowed. The value cannot start or end with the hyphen (-).

      OS

      Indicates the OS in use.

      Product Asset Label

      Indicates the product asset label. You can click Edit to modify it.

      The value is a string of 1 to 255 characters,

      and can contain digits, letters, and punctuation marks.

      Product SN

      Identifies a product.

      Firmware Version

      Indicates the current firmware version.

      MCU Version

      Indicates the current MCU version.

      NPU Version

      Indicates the current NPU version.

      Uptime

      Indicates the running time of the current system.

      Model

      Indicates the model of a peripheral.
    • View the information about peripherals. Table 2 describes the parameters.
      Table 2 Peripheral information parameters

      Parameter

      Description

      Product SN

      Identifies a product.

      Category

      Indicates the category of a peripheral.

      Vendor

      Indicates the vendor of a peripheral.

      NOTE:

      unknown indicates that the Atlas IES cannot identify the vendor name of the peripheral.

      Model

      Indicates the model of a peripheral.

      Health Status

      Indicates the health status of a peripheral.

      You can click to filter the health status of a peripheral. The options are as follows:

      • OK: normal
      • Critical: abnormal
      NOTE:

      After the filtering, you can click Reset to clear the filtering content.

      Startup Status

      Indicates the startup status of a peripheral.

      You can click to filter the startup status of a peripheral. The options are as follows:

      • Enabled: started
      • Disabled: stopped
        NOTE:

        After the filtering, you can click Reset to clear the filtering content.

      Device Location

      Indicates the location of the bus where a peripheral is located.

      Firmware Version

      Indicates the firmware version of a peripheral.
