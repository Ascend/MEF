Intelligent Edge System

The Atlas 500 A2 Intelligent Edge System (short for Atlas IES) is a system monitoring and management software for edge devices.

The Atlas IES has the following features:

  • Optimal user experience

    The Atlas IES provides a visualized and easy-to-use graphical user interface (GUI) for you to perform interactive operations on intelligent edge devices.

  • Efficient management and maintenance

    The Atlas IES provides remote management and hardware monitoring functions, which enable you to remotely access intelligent edge devices and monitor and manage device running status.

  • Secure access

    The Atlas IES provides various management interfaces, which use encryption algorithms to ensure secure system access.

This document covers device alarm monitoring, fault locating, system management, data configuration, and parameter description.
