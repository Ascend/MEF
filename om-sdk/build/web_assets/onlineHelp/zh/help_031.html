设置密码有效期

  1. 在主菜单中选择“设置 > 安全策略 > 密码有效期”。

    图1 密码有效期

  2. 设置有效期天数。

    可以通过单击列表框手动输入有效期天数。

    可以单击，设置有效期天数。每单击一次增加一天；每单击一次减少一天。

    图2 密码有效期设置
    • 取值范围为0～365，单位为天，默认值为90天；取值为0时表示密码有效期无限制；长期使用同一个密码会增加被暴力破解和恶意猜测的风险。
    • 为保障系统安全性，建议设置合适的密码有效期，并定期更新密码。

  3. 在“密码”文本框中输入系统当前的用户密码。
  4. 单击“保存”。

    界面弹出“保存成功”提示框，此时表示密码有效期设置成功。
