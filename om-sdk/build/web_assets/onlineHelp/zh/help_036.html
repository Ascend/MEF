恢复出厂

使用场景

设备运行过程中，当系统配置等信息发生损坏时，可以使用恢复出厂设置功能，使设备恢复至出厂状态。

  • 恢复出厂设置将会删除所有配置信息。此操作不可恢复，请谨慎操作。
  • 恢复出厂设置过程中，请保持供电稳定，如果异常断电，可能会导致恢复出厂设置失败。
  • 在固件升级过程中以及重启生效前，不支持执行恢复出厂设置操作。
  • 恢复出厂设置后，用于本地Web登录的IP地址默认只有一个。如果恢复出厂设置时选择保留IP地址，则恢复出厂设置后，保留的IP地址可用于本地Web登录；如果选择不保留IP地址，恢复出厂设置后，优先网口0的IP地址用于Web登录。
  • M.2通用系统上进行恢复出厂操作，恢复的是eMMC系统，重启后从eMMC启动。

操作步骤

  1. 在主菜单中选择“设置 > 恢复出厂”。

    进入“恢复出厂”操作页面。

    图1 恢复出厂

  2. 选择是否保留网络IP地址进行下一次登录。

    • 是，在“IP地址”处单击“保留”，执行3。
      在下拉框中选择需要保留的IP地址，以便下次登录使用。

      推荐用户选择保留当前登录系统的IP地址。若选择保留的IP地址与默认网关不在同一网段，恢复出厂设置后，会出现保留的IP地址无法远程登录系统的风险。

    • 否，在“IP地址”处单击“不保留”，执行4。

      不保留网络IP地址，网络IP地址将恢复为系统默认IP地址，可能无法远程访问设备。

  3. 选择保留IP地址。
  4. 在“密码”文本框中输入当前系统的root用户密码。
  5. 单击“保存”。

    界面弹出提示对话框，选择“确认”。

    恢复出厂过程中，请耐心等待6分钟左右。恢复出厂成功后，请手动刷新界面返回至系统登录页面。
