Log Collection

Scenarios

Collect logs on this page.

Procedure

  1. Choose Management > Log Collection.

    Figure 1 Log collection

  2. Click Collect and Download to collect logs. The system packs and downloads the collected logs.

    Figure 2 Collecting logs

Table 1 Log description

File Name

Description

MCU.tar.gz

MCU logs.

NPU.tar.gz

Current NPU logs.

OS_Drivers.tar.gz

OS_Driver logs.

mef_edge.tar.gz

MEF logs.

MindXOM.tar.gz

MindXOM logs.

collect_oper.tar.gz

Debug logs. collect_oper.tar.gz collects logs that are not collected by the container.

network-scripts.tar.gz

System network configuration file logs. network-scripts.tar.gz collects logs that are not collected by the container.

When logs are dumped, the date and time are added to the log file name.
