Security Policies

Choose Settings > Security Policy. Then, you can:

  • View and set the web timeout interval of the Atlas IES.

  • View and set the password validity period for the Atlas IES user.

  • View and manage user login security rules of the Atlas IES.

  • Import, export, or delete weak passwords of the Atlas IES.

  • Import, replace, and query the SSL server certificate.

  • View and set the number of days to generate an alarm in advance before the Atlas IES certificate expires.

For security purposes, you are advised to change the IP address listened by Nginx to a fixed IP address and disable Nginx from listening to IPv6 addresses after Nginx is successfully deployed.
