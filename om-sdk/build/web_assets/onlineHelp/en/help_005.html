User Login

Prerequisites

  • Google Chrome 69 or later is required.
  • HTTP is an insecure protocol. Use HTTPS for login.

Precautions

  • To ensure system security, change the initial password as prompted upon the first login and periodically change the password.
  • If you log in to the system for the first time after the firmware update, press F5 to refresh the page.
  • The system locks your account if you enter incorrect passwords (but correct account) for consecutive five times. You can log in again 5 minutes later. If the account you enter is incorrect, the system displays a message indicating that the user name or password is incorrect but does not lock the account.
  • If you forget the password, you can reset it by recovering factory settings. For details, see ies_tool restore_factory. You can also reset the Atlas IES account password on the FusionDirector managing the Atlas 500 A2 edge station. For details, see resetting device passwords in "Device Management > Managing Connected Devices > Edge Device Management" in the FusionDirector Operation Guide.
  • By default, the system timeout period is 15 minutes. If you do not perform any operation on the WebUI for 15 minutes, you will be automatically logged out of the WebUI.

Procedure

This section uses a PC running Windows 10 and Google Chrome 81.0 as an example.

  1. Connect cables and deploy the network environment.

    1. Use an Ethernet cable to connect the PC to the network port whose Function is set to web of the Atlas 500 A2 edge station.
    2. Configure the network port IP address and subnet mask of the PC, so that the PC can access the Atlas 500 A2 edge station.

    You are advised to set the network port IP address of the PC and the management network port IP address of the Atlas 500 A2 edge station in the same network segment. If the IP addresses are not in the same network segment, configure a router to ensure that the PC can correctly communicate with the edge station.

  2. Open Google Chrome, enter https://XXX.XXX.XXX.XXX in the address box, and press Enter. The login page is displayed.

    • XXX.XXX.XXX.XXX indicates the IP address for accessing the Atlas IES WebUI. The default IP address of network port 0 is 192.168.2.111, and the default IP addresses of network ports 1â€“4 are 192.168.3.111.
    • If a message indicating a security certificate problem of the website is displayed, click Continue to this website (not recommended). The browser page varies depending on the Google Chrome version.
    Figure 1 Logging in to Atlas IES.

  3. Select a language.

    Figure 2 Selecting a language

  4. Enter the user name and password. For details about the user name and password, see "Atlas 500 A2 Edge Station" in the Atlas Hardware Product User List.
  5. Click Log In.

    • Non-initial login: After you perform 5, the Atlas IES home page is displayed.
    • Initial login: After performing 5, perform 6 to 8 to go to the Atlas IES home page.

  6. In the displayed dialog box, enter the old password and new password, confirm the password, and click Confirm.

    Figure 3 Changing user password
    A password must meet the following requirements:

    • Contains 8 to 20 characters.

    • Contains at least three types of the following characters:

      • Lowercase letters: a to z

      • Uppercase letters: A to Z

      • Digits 0 to 9

      • Special characters: `~!@#$%^&*()-_=+\|[{}];:'",<.>/? and space
    • Cannot be the same as the user name or user name in reverse order.
    • Must be different from the last five passwords and cannot be the default password.

  7. On the login page, enter the user name and new password again.
  8. Click Log In.

    After the login is successful, the home page is displayed.
