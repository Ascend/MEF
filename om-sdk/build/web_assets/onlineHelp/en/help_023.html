Firmware Update

The following describes how to update the firmware on the Atlas IES.

Obtaining the Update Packages

Before updating the software, obtain the software packages and digital signature files listed in the following table.

Table 1 Required software packages

Software Package

Description

How to Obtain

Ascend-hdk-310b-npu-soc_{version}_linux-aarch64.zip

Update package of the Atlas 500 A2 edge station, which is used to update the NPU firmware.

Link

Ascend-hdk-310b-mcu_{version}.zip

Update package of the Atlas 500 A2 edge station, which is used to update the MCU firmware.

A500-A2-firmware_{version}.zip

Update package of the Atlas 500 A2 edge station, which is used to update the device OM and firmware.

{version} indicates the package version. Download the software package as required.

Verifying the Digital Signature

To prevent a software package from being maliciously tampered with during transfer or storage, download also the corresponding digital signature file for integrity verification while downloading the software package.

After the software package is downloaded from the Huawei Support website, verify its PGP digital signature. For details, see the OpenPGP Signature Verification Guide. If the verification fails, do not use the software package, and contact Huawei technical support.

The verification is also required before the installation or update of the software package.

For carriers, visit https://support.huawei.com/carrier/digitalSignatureAction.

For enterprise users, visit https://support.huawei.com/enterprise/en/tool/pgp-verify-TL1000000054.
