Switching to Root Login (develop)

Function

The develop command is used to switch to the root user for login.

Syntax

develop

Parameter Description

No parameters are supported. The system prompts you to enter the password of the root user.

Usage Guidelines

For details about commands used after the develop command is executed, see common Linux commands.

Example

IES:/->develop
Command output:
input root passwd
Password:
Last login: Tue Jun  4 15:38:59 UTC 2019 on pts/0 
Euler:~ #
