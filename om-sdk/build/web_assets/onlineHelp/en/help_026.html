Reset Control

Scenarios

Restarting the system affects ongoing user services. Forcibly power cycling the system may damage system data. During routine maintenance, perform operations as required.

Restart System

  1. Choose Management > Reset Control.

    Figure 1 Reset control

  2. Click Restart. A confirmation dialog box is displayed.

    Figure 2 Restarting the system

    After you click Restart, the system is forcibly restarted. This operation affects user services that are running.

  3. Click Confirm. Wait until the system is restarted.

Forced Power Cycle

  1. Choose Management > Reset Control.

    Figure 3 Forced power cycle

  2. Click Forced Power Cycle. A confirmation dialog box is displayed.

    Figure 4 System restart

    After you click Forced Power Cycle, the system will be forcibly powered off and then powered on. This operation may damage system data. Exercise caution when performing this operation.

  3. Click Confirm.

If the Docker service fails to be started after forcible power-off and power-on, rectify the fault by referring to Docker Service Fails to Be Started Occasionally After the System Is Forcibly Powered Off and Powered On.
