# 欢迎使用 MEF

<div style="text-align: center; margin: 0.5rem 0 0.3rem 0; font-family: 'Avenir Next', 'Avenir', 'Century Gothic', 'Segoe UI', sans-serif;">
  <span style="font-size: 4.5rem; font-weight: 300; letter-spacing: 0.02em;">MEF</span>
</div>

MEF是昇腾推理解决方案中一款轻量化端边云协同使能框架，提供边缘节点管理和智能推理业务（容器应用）管理的功能。可通过MEF Edge和MEF Center进行边云协同管理，用户可通过二次开发，对接ISV（Independent Software Vendor）业务平台，集成所需功能。本docs仓提供MEF的相关指导文档。
